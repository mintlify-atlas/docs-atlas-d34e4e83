## What is Start UI [web]?

Start UI [web] is an opinionated frontend starter repository created and maintained by the [BearStudio Team](https://www.bearstudio.fr/team). It represents their up-to-date stack for building modern web applications with a focus on developer experience, type safety, and production readiness.

This starter comes with authentication, database integration, a comprehensive UI component library, email templates, testing infrastructure, and deployment configurations out of the box.

## Technology Stack

Start UI [web] leverages the latest and most reliable technologies:

### Core Framework
- **[TanStack Start](https://tanstack.com/start)** - Full-stack React framework with file-based routing
- **[React 19](https://react.dev/)** - Latest React with React Compiler support
- **[TypeScript](https://www.typescriptlang.org/)** - Full type safety across the stack
- **[Node.js 22+](https://nodejs.org)** - Modern Node.js runtime

### Backend & Data
- **[Prisma](https://www.prisma.io/)** - Next-generation ORM for PostgreSQL
- **[oRPC](https://orpc.unnoq.com/)** - Type-safe RPC framework with OpenAPI support
- **[Better Auth](https://www.better-auth.com/)** - Flexible authentication library
- **[PostgreSQL](https://www.postgresql.org/)** - Robust relational database

### UI & Styling
- **[Tailwind CSS v4](https://tailwindcss.com/)** - Utility-first CSS framework
- **[shadcn/ui](https://ui.shadcn.com/)** - Re-usable component library
- **[Lucide Icons](https://lucide.dev/)** - Beautiful icon set
- **[next-themes](https://github.com/pacocoursey/next-themes)** - Dark mode support

### Developer Experience
- **[Storybook](https://storybook.js.org/)** - Component development and documentation
- **[Vitest](https://vitest.dev/)** - Fast unit testing
- **[Playwright](https://playwright.dev/)** - Reliable E2E testing
- **[ESLint](https://eslint.org/)** - Code quality and consistency
- **[Prettier](https://prettier.io/)** - Code formatting
- **[Lefthook](https://github.com/evilmartians/lefthook)** - Fast Git hooks

### Infrastructure
- **[Docker](https://www.docker.com/)** - Containerized development environment
- **[MinIO](https://min.io/)** - S3-compatible object storage
- **[Maildev](https://maildev.github.io/maildev/)** - Email testing in development

## Requirements

Before you begin, ensure you have the following installed:

- **Node.js** >= 22
- **pnpm** package manager
- **Docker** (or a PostgreSQL database)

<Note>
  Start UI [web] uses pnpm for package management. Make sure you have it installed globally:
  ```bash
  npm install -g pnpm
  ```
</Note>

## Quick Start

Get started with Start UI [web] in seconds:

```bash
pnpm create start-ui -t web myApp
```

This command scaffolds a new project with the latest version of Start UI [web].

## What's Included

Your new Start UI [web] project includes:

- **Authentication flows**: Login, signup, email verification, and logout
- **User management**: Admin panel with user roles and permissions
- **Dashboard**: Sample dashboard with data visualization
- **CRUD examples**: Book management feature demonstrating best practices
- **Form components**: Validated forms with error handling
- **File uploads**: Image and file upload with preview
- **Email templates**: Transactional email templates with i18n
- **API documentation**: Auto-generated OpenAPI docs
- **Internationalization**: Multi-language support (English and French)
- **Dark mode**: Automatic and manual theme switching
- **Responsive design**: Mobile-first responsive layouts
