## Overview

Start UI [web] is a modern full-stack React application built with cutting-edge technologies for building production-ready web applications. It follows a feature-driven folder structure and leverages type-safety throughout the entire stack.

## Tech Stack

### Core Framework

- **TanStack Start** - Full-stack React framework with file-based routing
- **React 19** - UI library with latest features
- **TypeScript** - Type-safe development
- **Vite** - Fast build tool and dev server

### Backend

- **Better Auth** - Modern authentication library with OAuth and email OTP support
- **Prisma ORM** - Type-safe database client with PostgreSQL
- **oRPC** - Type-safe RPC framework for API layer
- **Pino** - High-performance logging

### Frontend

- **TanStack Router** - Type-safe file-based routing
- **TanStack Query** - Server state management
- **React Hook Form** - Form state management
- **Zod** - Schema validation
- **Tailwind CSS 4** - Utility-first CSS framework

### Developer Tools

- **Vitest** - Unit and component testing
- **Playwright** - E2E testing
- **Storybook** - Component development
- **ESLint & Prettier** - Code quality and formatting

## Feature-Driven Organization

Each feature in the `features/` directory is self-contained with:

- **Page components** - Split into `app/` (user-facing) and `manager/` (admin)
- **Schemas** - Zod validation schemas
- **Types** - TypeScript types and interfaces
- **Utilities** - Feature-specific helpers

## Server-Side Architecture

The server layer is organized around oRPC procedures:

1. **Base procedures** (`server/orpc.ts`) - Define middleware chain
2. **Route handlers** (`server/routers/*`) - Implement business logic
3. **Router composition** (`server/router.ts`) - Combine all routes
4. **API endpoint** (`routes/api/rpc.$.ts`) - HTTP handler

## Key Design Principles

1. **Type Safety** - End-to-end type safety from database to UI
2. **Feature Isolation** - Features are self-contained modules
3. **Code Colocation** - Keep related code together
4. **Separation of Concerns** - Clear boundaries between layers
5. **Developer Experience** - Fast feedback loops with hot reload
